# ThinkBoard 📝

A full-stack note-taking application built with the MERN stack.

## Live Demo

Frontend: https://mern-thinkboard-delta.vercel.app

## Features

- Create Notes
- Edit Notes
- Delete Notes
- Responsive UI
- MongoDB Atlas Integration
- Upstash Redis Rate Limiting
- MERN Stack Architecture

## Tech Stack

- React.js
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- Upstash Redis
- Tailwind CSS
- Axios

## What I Learned

Through building this project, I gained hands-on experience with:

- Building REST APIs using Express.js
- MongoDB integration with Mongoose
- React component-based development
- CRUD operations (Create, Read, Update, Delete)
- Environment variable management with .env
- Rate limiting using Upstash Redis
- Frontend and backend communication using Axios
- Deploying a MERN application using Vercel and Render
- Handling CORS issues in production
- Working with Git and GitHub for version control

## Installation

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
